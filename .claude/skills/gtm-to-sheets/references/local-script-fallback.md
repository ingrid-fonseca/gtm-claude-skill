# Local Script Fallback

Use this when the sandboxed environment can't reach the Google APIs.
The skill will generate this script and the user runs it on their own machine.

---

## What to tell the user

> "The network here can't reach Google's APIs directly. I'll generate a
> self-contained Python script you can run on your own machine — it only needs
> Python 3.8+ and the `requests` library."

---

## Script template

When generating the fallback script, Claude should:

1. Embed the parsed rows as a Python list-of-lists literal (already computed).
2. Fill in the access token placeholder.
3. Include all three API calls (create sheet, write values, format header).

```python
#!/usr/bin/env python3
"""
GTM → Google Sheets export script
Generated by Claude. Run with: python gtm_to_sheets.py
Requires: pip install requests
"""

import requests

ACCESS_TOKEN = "PASTE_YOUR_TOKEN_HERE"
CONTAINER_NAME = "{container_name}"

HEADERS = {
    "Authorization": f"Bearer {ACCESS_TOKEN}",
    "Content-Type": "application/json",
}

ROWS = [
    ["Tag Name", "Platform / Vendor", "Tag Type", "Triggers", "Status"],
    # --- data rows inserted here ---
    {rows_placeholder}
]

# 1. Create spreadsheet
r = requests.post(
    "https://sheets.googleapis.com/v4/spreadsheets",
    headers=HEADERS,
    json={"properties": {"title": f"GTM Container Audit – {CONTAINER_NAME}"},
          "sheets": [{"properties": {"title": "Tags"}}]},
)
r.raise_for_status()
sheet_id = r.json()["spreadsheetId"]
print(f"Created sheet: https://docs.google.com/spreadsheets/d/{sheet_id}/edit")

# 2. Write data
r = requests.put(
    f"https://sheets.googleapis.com/v4/spreadsheets/{sheet_id}/values/Tags!A1",
    headers=HEADERS,
    params={"valueInputOption": "RAW"},
    json={"values": ROWS},
)
r.raise_for_status()

# 3. Format header
r = requests.post(
    f"https://sheets.googleapis.com/v4/spreadsheets/{sheet_id}:batchUpdate",
    headers=HEADERS,
    json={"requests": [
        {"repeatCell": {
            "range": {"sheetId": 0, "startRowIndex": 0, "endRowIndex": 1},
            "cell": {"userEnteredFormat": {
                "backgroundColor": {"red": 0.27, "green": 0.51, "blue": 0.71},
                "textFormat": {"bold": True,
                               "foregroundColor": {"red": 1, "green": 1, "blue": 1}},
            }},
            "fields": "userEnteredFormat(backgroundColor,textFormat)",
        }},
        {"autoResizeDimensions": {
            "dimensions": {"sheetId": 0, "dimension": "COLUMNS",
                           "startIndex": 0, "endIndex": 5},
        }},
    ]},
)
r.raise_for_status()
print("Done! Sheet formatted and ready.")
```
